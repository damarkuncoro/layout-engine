# Examples

## Headless
- Jalankan build paket lalu:
  - `node ../packages/layout-engine/dist/tests/basic.test.js`
- Contoh headless ada pada root:
  - `examples/headless-basic.mjs`
  - `examples/headless-dashboard.mjs`

## React SSR
- Lokasi: `examples/react-ssr`
- Perintah:
  - `cd examples/react-ssr`
  - `npm install`
  - `npm start`

## React SPA (Vite)
- Lokasi: `examples/react-spa`
- Perintah:
  - `cd examples/react-spa`
  - `npm install`
  - `npm run dev`
- Fitur:
  - Dashboard dengan sidebar collapsible
  - Grid demo (static, masonry, areas)
  - Grid areas responsif (layout berubah sesuai lebar)
  - Responsive demo (gap & columns)
  - Flex demo (direction & gap)
  - Responsive sidebar width
  - Responsive Box (padding/margin)
  - Dark mode toggle
  - Router demo (nested layout & params + breadcrumbs)
